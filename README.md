# IDM Activation Script v1.9.5

> By Nadif Rizky

> **Alat aktivasi sekali klik untuk Internet Download Manager (IDM)**: mendukung pembekuan masa uji coba IDM, aktivasi dengan informasi registrasi acak, dan reset masa uji coba sekali klik. Menu dan prompt sudah diterjemahkan dengan jelas. Berjalan di Windows 7 / 8 / 10 / 11.

## Unduh Cepat

> **Unduhan yang disarankan**: buka halaman GitHub repo ini lalu gunakan **Releases** bila tersedia, atau **Code > Download ZIP** untuk mengunduh isi repo langsung.

## Fitur

- Mendukung versi IDM terbaru
- Tiga mode: aktivasi beku, aktivasi biasa, dan reset
- Pencadangan otomatis registri sebelum perubahan
- Tidak memodifikasi file program IDM
